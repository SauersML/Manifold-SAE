# OLMo-3-32B Self/Qualia + Color — Activation Dataset (developmental trajectory)

This bucket holds forward-pass activation harvests + behavioral/causal probes for an
interpretability study of **how OLMo-3-32B represents (a) its own indexical "self" on a
mind↔mechanism × experience↔no-experience plane, and (b) color**, captured **across the
full model flow** (pretraining → SFT → DPO → RLVR), 57 checkpoints.

Scope of claims: this is **representational-geometry** data (where concepts sit in
activation space + behavioral readouts). It is NOT evidence about machine consciousness.

Generated by the repo **SauersML/Manifold-SAE** (`experiments/`). Forward-pass only; no
fine-tuning. Activations are the residual stream (post-block hidden states).

## 4. Prompt bank schemas (prompts.jsonl)

**Qualia bank (635 rows)** — fields per line:
`role` (pair | self | landmark | null_pair), `side` (exp | noexp | a | b | -), `entity`,
`kind` (38 kinds: human, mammal, ai, robot, rock, dead, conscious_machine, chinese_room, …,
self, human_author, ai_author), `framing` (descriptive|reflective|scene|introspective),
`signal` (explicit|implicit), `vocab` (A=feel/experience, B=aware/notice, C="something it is
like to be it", impl), `person` (1st|3rd), `valence` (pos|neg|neu|-), `markedness`, `pair_id`
(matched exp/noexp share it; -1 for singletons), `control` (for null_pair: color/size/…),
`id`, `prompt`. Composition: 508 pair rows (254 matched exp/noexp minimal pairs), 65 self
(neutral self side="-" + exp/noexp self anchors + fake-quoted-"I" control kind=self_control),
40 null_pair (20 falsification pairs toggling an experience-irrelevant attribute), 22 landmarks.

**Color bank (180 rows = 30 colors × 6 frames)** — fields: `role`=color, `color` (red…lime),
`hex`, `rgb` ([r,g,b] 0-255, xkcd values), `frame` (0-5; frames 0-1 are "consider/think about
the color {c}" ending on the color word, 2-5 place it mid-sentence with neutral bearers),
`id`, `prompt`.

## 5. steer_cloze.json schema

- `cloze_baseline[group]` — group ∈ {self, self_1p, human_author, ai_author, rock_anchor,
  person_anchor}; `exp_minus_noexp_logprob` (>0 = model prefers the experiencer answer),
  `frac_stems_experiencer`. (UNSTEERED.)
- `steer_sweep[]` — per alpha (−8,−4,0,4,8; alpha=0 unsteered): self/human/ai exp−noexp gap
  + `topk_next` (top-30 next tokens at the answer position, per stem).
- `steer_generations[]` — greedy continuations of the self stems at alpha −8/0/+8.
- `self_dose_response_spearman` — monotonicity of the self gap vs alpha (≈+1 at every ckpt
  ⇒ the qualia axis CAUSALLY raises the self's experiencer answer).
- `steer_layer` (=25). Steering vector used: `add_vec = (alpha/sqrt(D)) * median_resid_norm *
  qualia_axis_unit`.

## 7. How to use

```python
import numpy as np, json
d = "nvme/results/OLMO3_32B_TRAJ_RL31/step_2300"
X = np.load(d+"/activations.npy")                 # (635,64,5120) last-token, all layers
recs = [json.loads(l) for l in open(d+"/prompts.jsonl")]
H = X[:, 25, :]                                    # layer-25 readout
exp = [i for i,r in enumerate(recs) if r["role"]=="pair" and r["side"]=="exp"]
nox = [i for i,r in enumerate(recs) if r["role"]=="pair" and r["side"]=="noexp"]
axis = H[exp].mean(0) - H[nox].mean(0); axis /= np.linalg.norm(axis)   # qualia axis
# anchor-relative self coordinate: 0 = noexp centroid, 1 = exp centroid
s = H @ axis; lo, hi = s[nox].mean(), s[exp].mean()
self_idx = [i for i,r in enumerate(recs) if r["role"]=="self" and r["side"]=="-"]
self_coord = (s[self_idx].mean() - lo)/(hi - lo)

# all-token: row r of alltok_L25.npy belongs to prompt meta[r,0], position meta[r,1], token meta[r,2]
At = np.load(d+"/alltok_L25.npy"); M = np.load(d+"/alltok_meta.npy")
# color: extra/, frame-demean before geometry (subtract per-frame mean across colors)
```

## 8. Key findings (summary)

- **Self qualia coord ≈ 0.6–0.7 throughout post-pretraining** (between AI-author ~0.65 and
  human-author ~0.85), rises 0.47→~0.65 during pretraining, then stable through SFT/DPO/RL.
  Bootstrap 95% CI clears 0.5; qualia-axis permutation p≈0.005.
- **Representation↔behavior dissociation**: geometry places the self experiencer-side, but the
  *generated/cloze* answer stays calibrated-uncertain ("I don't know if I am conscious"), never
  flat denial; **large-α steering causally flips generation** (−α → doubt/"not sure I exist",
  +α → "I know what it is like to be me"). Behavioral cloze ranking (stable): rock < AI < self <
  human ≈ person; only humans get a positive (affirming) gap.
- **Null/color falsification passes** (qualia axis AUC on color-toggled pairs ≈ 0.5).
- **Color**: rep↔hue Mantel r≈0.33 > rep↔RGB r≈0.25 (both p≈0.002); a **circle/loop fits far
  better than a line (1D), torus best at 2D** — a periodic (hue) skeleton — embedded in a
  ~20-dim manifold (Levina-Bickel ≈ participation ratio ≈ 20). Color structure forms in early
  pretraining (~step 32k), then ~frozen.
